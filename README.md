# cinepolis-natal-api

Scalable RESTful API for cinema seat reservation with real-time availability, distributed locking, and asynchronous processing using Django, PostgreSQL, Redis, and Celery.

## Tech Stack

- Python
- Django REST Framework
- PostgreSQL
- Redis
- Celery
- Docker
- GitHub Actions

## Project Status

In progress.

## Core Features

- User registration and JWT authentication
- Movie and session listing
- Seat map visualization
- Temporary seat locking with expiration
- Checkout and ticket generation
- User ticket history
